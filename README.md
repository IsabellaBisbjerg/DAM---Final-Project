# Sentiment, Gender and Correspondence during the First Schleswig War

## Description
This projct researches the sentimental developement for both women and men around the time of the First Schleswig War. We have made sentiment analysis of a list of letters conducted by members of the upper class in Denmark in the middle of the 19th century
